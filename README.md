# Syntax Station

A simple, open-source reference website for programming language syntax. Available at: [syntaxstation.com](https://syntaxstation.com)

## Overview

Syntax Station provides a quick and easy way to review fundamental syntax for various programming languages. The site features a modern dark theme, responsive design, and syntax highlighting powered by Prism.js.

**Supported Languages:**
* C
* C++
* C#
* Java
* JavaScript
* Python
* Lua
* Perl
